P2P Lending Analytics & Credit-Risk Modeling

Reproducible machine learning pipelines for credit risk assessment on Bondora P2P lending data

Funded by: Swiss National Science Foundation (Grant #205487)

O Overview

This project develops and evaluates machine learning approaches for credit-risk modeling in peer-to-peer (P2P) lending markets. Using comprehensive data from the Bondora lending platform, we build interpretable models for default prediction and risk segmentation.

The research emphasizes reproducibility through documented notebooks, modular code, and transparent methodology that enables other researchers to validate and extend our findings.

Credit Risk P2P Lending Machine Learning Python Reproducible Research

M Methodology

  • Feature engineering from loan and borrower characteristics
  • Graph-based credit scoring with borrower similarity networks
  • Supervised classification for default prediction
  • Model interpretability using SHAP and feature importance
  • Cross-validation and out-of-sample evaluation
  • Risk segmentation and portfolio analysis
